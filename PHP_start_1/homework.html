Каждое задание выполнить в отдельном файле. Имя файла должно соответствовать номеру задачи. Файлы с решениями разместить в своем репозитории git

  1. Создайте переменную $name и присвойте ей строковое значение, содержащее Ваше имя. Создайте переменную $age и присвойте ей числовое значение, содержащее Ваш возраст. С помощью оператора echo или print выведите две строки: "Меня зовут <Ваше имя>", "Мне <Ваш возраст> лет". Каждая фраза должна начинаться с новой строки
  2. Создайте переменную $age и присвойте ей случайное число от 0 до 100 (воспользуйтесь функцией rand(0, 100)).
    Скрипт должен вывести фразу "Возраст <значение переменной $age>" и на новой строке:
    1. "Вам еще работать и работать" если значение $age лежит в интервале от 18 до 65 включительно
    2. "Вам пора на пенсию" если значение $age больше 65
    3. "Вам еще рано работать" если значение $age меньше 18
  3. Есть код:
        <?php
    $a = 1;
    if('2' == $a){
        echo 'TEXT 1';
    }
    if('3' == $a){
        echo 'TEXT 2';
    }
    Измените строку "$a = 1;" таким образом, чтобы выполнились оба условия. Изменять, дополнять или удалять другие строки нельзя.
    Подсказка: задача на приведение типов
